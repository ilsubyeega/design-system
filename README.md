# design-system
본 디자인 시스템은 PC, 모바일 및 게임 내에서 사용하기 위해 개발된 디자인 시스템입니다. 이 시스템은 일관된 디자인 원칙과 스타일을 적용하여 사용자 경험을 향상시키고, 프로젝트 내에서 일관성과 효율성을 유지하는 데 목적을 두고 있습니다.
